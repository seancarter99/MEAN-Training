Monday Topics

HTML
- html = hypertext markup language
- why use markup?
  - rendering
  - search
- hypertext = links between pages
- structure = hierarchy/tree
- tag = wrap content & gives meaning
  - ex: <p>content</p>
- meta tags = information about our page, not rendered to the page
- ex: title, author, keywords (SEO), icon



- block elements - take the entire width, always start on the next line
- inline elements - size is just big enough for its contents



- semantic tags = give more meaning to the sections of the document

CSS
----
3 ways to apply styles
  1. inline (use attribute style="")
  2. internal (use tag <style></style>)
  3. external (use external style.css and <link rel="stylesheet" href="style.css">)
selectors
- tag names
- id's use #idname
- classes use .clasname
- pseudo classes use :classname ex: hover
- relative vs absolute units
  - use absolute for font and border
  - use relative the rest the time
properties
- color
- background-color
- display: inline (or block or none or flex etc.)
- width
- height
- margin
- padding
